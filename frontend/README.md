# 🏥 SMARTMED - Healthcare Management System

SMARTMED is a modern, responsive Single Page Application (SPA) designed to simplify patient-doctor interactions. It features a secure dashboard, appointment booking system, and AI-powered health assistance.

## ✨ Key Features

### 🔒 Security & Authentication
- **Protected Routes:** Dashboard and private pages are secured with a "Guard" system; unauthenticated users are redirected.
- **Simulated Auth:** Complete Login/Signup flow with session management using Local Storage.
- **Ticket System:** Secure logout functionality that clears session tokens.

### 🩺 Patient Dashboard
- **Find Doctors:** Browse specialist profiles with ratings and experience.
- **Appointment System:** Book appointments that persist to a "History" page (Data Persistence).
- **Medical Reports:** Access mock medical records.
- **AI Chatbot:** Interactive chat interface for immediate health queries.

### 🎨 Modern UI/UX
- **Dual Theme:** Fully functional **Dark Mode** 🌙 and "Platinum Blue" Light Mode ☀️.
- **Responsive Design:** Mobile-first architecture that adapts to phones, tablets, and desktops.
- **Interactive Elements:** Toast notifications, smooth transitions, and dynamic profile editing.

## 🛠️ Tech Stack
- **Frontend:** React.js (Vite)
- **Routing:** React Router DOM (v6)
- **Styling:** CSS Modules & Variables
- **Icons/Notifications:** React Hot Toast
